# Actividad 1. Creación de una página web básica con HTML: Aplicación de estructura y etiquetas de texto
Objetivo: Crear una página web sencilla en Visual Studio Code que ponga en práctica todos los conceptos enseñados.

# Instrucciones:
Crear una estructura básica de HTML:

Comenzar con la estructura principal (<!DOCTYPE html>, <html>, <head>, <body>).

# Título y Metadatos:
En el <head>, incluir un <title> que describa el contenido de la página.
Añadir un comentario que explique cuál es la función del <head>.

# Encabezado (<header>):
Crear un <header> que contenga el nombre del proyecto como título principal usando una etiqueta de encabezado (<h1>).
Añadir un subtítulo debajo del título usando una etiqueta de menor jerarquía (<h2>).

# Sección de Contenido Principal:
Añadir al menos tres párrafos (<p>) con información ficticia.
Dentro de los párrafos:

Utilizar etiquetas de texto como <b>, <strong>, <em>, <i>, <u>, <big>, <small>, <sub>, <sup>, <ins>, <del>, y <mark> para resaltar partes del texto. Cada párrafo debe contener al menos tres de estas etiquetas.
Incluir un salto de línea (<br>) en uno de los párrafos.
Añadir una línea horizontal (<hr>) para separar el contenido.

# Enlaces:
Crear una lista de tres enlaces utilizando la etiqueta <a> con el atributo href.
Los enlaces deben apuntar a cualquier sitio web externo (ejemplo: https://www.google.com).
Añadir un atributo target="_blank" para que se abran en una nueva pestaña.

# Aplicar id y class:
Asignar un id único (ejemplo: id="parrafo1", id="parrafo2").
Crear una clase (class) llamada resaltado y aplicarla a dos de los párrafos para identificar que comparten el mismo estilo.

# Footer (<footer>):
Crear un <footer> con un párrafo (<p>) que contenga un comentario indicando que es el final de la página.
